# Philippine Real Estate EDA

Exploratory Data Analysis of 1,496 property listings across the Philippines.

---

## Key Questions Answered

- What does a typical property look like?
- Does size predict price?
- Which cities are cheapest per sqm?
- Do bathrooms and bedrooms affect price?
- What's considered a good deal?

---

## Key Insights

- **Most properties are under ₱10M** – average is skewed by luxury estates
- **Size is the #1 price driver** – 1% bigger home = 1.24% higher price
- **Location matters more than size** – where you buy often beats how big
- **Price per sqm is the best value metric** – more space for less money

---

## Tools Used

- Python (pandas, numpy, matplotlib, seaborn, scipy, statsmodels)
- Jupyter Notebook
- Power BI
